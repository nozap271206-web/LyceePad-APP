# LycéePad

## 📌 Présentation du projet
**LycéePad** est une application web de visite guidée interactive du Lycée Polyvalent Saint-Éloi d’Aix-en-Provence.  
Elle permet aux visiteurs (élèves, parents, partenaires) de découvrir les espaces du lycée de manière autonome grâce à des contenus multimédias géolocalisés et des parcours personnalisés.

Le projet s’inspire du principe des **HistoPad** (châteaux de la Loire).

---

## 🎯 Objectifs
- Proposer une visite interactive et autonome du lycée
- Valoriser l’établissement et ses formations
- Faciliter l’orientation des visiteurs
- Offrir des contenus adaptés à différents profils
- Fonctionner sur tablette et mobile (PWA)

---

## 🛠️ Technologies utilisées

### Frontend
- HTML / CSS / JavaScript
- **Vue.js**
- Vite / Webpack
- Leaflet.js (carte interactive)
- Responsive mobile-first

### Backend
- **Node.js**
- **Express.js**
- API REST
- **MySQL**

### PWA
- Service Worker
- Web App Manifest
- Mode hors-ligne
- HTTPS obligatoire

### Outils
- Git / GitHub
- Figma (maquettes)
- Draw.io (schémas UML)
- Lighthouse (audit PWA)
- Postman (tests API)

- Interface d’administration des contenus
- Mode hors-ligne
